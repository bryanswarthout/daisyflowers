## What We Built
A voice-enabled AI cannabis budtender web app that recommends products from Beyond Hello dispensary.

## Key Features
- **AI Chatbot**: Daisy Flowers character using Anthropic Claude
- **Voice Interaction**: Speech-to-text input + text-to-speech output
- **Animated Avatar**: CSS-animated character with speaking animations
- **Product Recommendations**: Smart filtering of 10,000+ cannabis products
- **E-commerce Integration**: Direct links to Beyond Hello product pages

## Tech Stack
- **Backend**: Node.js + Express + iHeartJane API
- **Frontend**: React + Vite
- **AI**: Anthropic Claude 3.7
- **Voice**: Web Speech API

## Product Filtering
- **6 Approved Brands**: Hijinks, Lab, Nira+, Flower Foundry, Seche, Tasteology
- **4 Categories**: Flower, Edibles, Vapes, Concentrates
- **Smart Search**: Natural language processing for cannabis terms

## Compliance
- Medical disclaimers on all recommendations
- No therapeutic claims
- Brand-restricted product catalog
